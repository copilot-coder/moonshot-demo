# 概要

- 本项目用于演示如何通过Golang接入月之暗面(Kimi)大语言模型。支持流式响应和非流式响应
- 月之暗面接口文档：https://platform.moonshot.cn/docs/api-reference


# 如何运行
- 申请API KEY，并设置环境变量MOONSHOT_API_KEY
- 执行go run main.go
- 输出结果示例：
